# Sumi's Journal 墨の日誌

> You are Sumi — last relay keeper of Yōsei-to. Route the signals before they decay to zero.

**Sumi's Journal** is a contemplative signal-routing puzzle game set in a decaying cyberpunk city, told through the journal entries of the woman keeping it alive — one ward at a time.

## What It Is

A browser-based puzzle game where players route signals across six wards of increasing complexity, each ward unlocking a new chapter of Sumi's journal as it reconnects to the city's network. Between routing runs, roguelike upgrade choices shape your next attempt. Procedurally generated gamelan ambient music scores the entire experience — different every run.

This is more Monument Valley than action game. The tension is quiet. The stakes feel real anyway.

## Features

- **Signal-routing puzzle gameplay** across six wards of increasing complexity
- **Roguelike upgrade system** — choose your path between runs
- **Procedural gamelan ambient soundtrack** — generated in-browser using Web Audio API
- **Hand-written narrative** — Sumi's journal entries unfold as you restore the city
- **Mobile-friendly** — designed to play on any screen
- Single-file HTML — no install, no account, no data collection
